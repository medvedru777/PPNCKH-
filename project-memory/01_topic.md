# Topic 15

## Title

Quantitative Survey on Explainable Features in Malware Machine Learning

---

## Core Direction

This project focuses on explainability in malware detection models instead of accuracy-only optimization.

The study investigates:
- PE Header feature importance
- API Import feature importance
- SHAP interpretation
- false positive behavior
- explainability reliability

---

## Research Goal

Understand why the model makes decisions rather than only measuring predictive performance.
